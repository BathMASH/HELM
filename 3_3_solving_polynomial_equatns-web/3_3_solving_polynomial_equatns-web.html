Solving Polynomial Equations

Introduction

Linear and quadratic equations, dealt within Sections 3.1 and 3.2, are members of a class of equations, called polynomial equations . These have the general form:

a n x n + a n − 1 x n − 1 + … + a 2 x 2 + a 1 x + a 0 = 0

in which x is a variable and a n , a n − 1 , … , a 2 , a 1 , a 0 are given constants. Also n must be a positive integer and a n ≠ 0 . Examples include x 3 + 7 x 2 + 3 x − 2 = 0 , 5 x 4 − 7 x 2 = 0 and − x 6 + x 5 − x 4 = 0 . In this Section you will learn how to factorise some polynomial expressions and solve some polynomial equations.

Prerequisites

  • be able to solve linear and quadratic equations

Learning Outcomes

  • recognise and solve some polynomial equations
